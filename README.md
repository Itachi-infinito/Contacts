# SparkWork 🚀

Application mobile de recrutement développée en .NET MAUI.

## 📱 Features
- Création et gestion d'offres d'emploi
- Chat en temps réel candidat / recruteur
- Swipe de candidats
- Profil recruteur modifiable

## 🛠️ Technologies
- .NET MAUI
- C#
- SQLite
